# portal-main-guide

## 项目简介

portal-main-guide 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面存放与索引的载体，方便对分散的 HTML 文档进行集中管理和版本控制。

## 目录说明

```
portal-main-guide/
├── pages/           # 存放所有独立 HTML 页面文件
├── assets/          # 页面所需的静态资源（图片、样式、脚本等）
├── index.html       # 仓库主页（可选，用于展示页面列表）
└── README.md        # 本文件
```

- `pages/`：每个 HTML 文件对应一个独立页面，文件命名建议简洁清晰。
- `assets/`：按需存放页面引用的公共或私有资源。
- `index.html`：可选的入口页面，用于汇总或导航。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接通过浏览器打开或部署到任何静态托管服务。
- 页面之间无强制关联，每个文件自包含其样式与逻辑。
- 归档时不限定页面主题或用途，仅确保文件结构整洁、命名合理。

## 维护说明

- 新增页面：将 HTML 文件放入 `pages/` 目录，并视情况更新 `index.html` 中的链接。
- 移除页面：直接删除对应文件，并清理相关引用。
- 资源管理：建议将跨页面共用的资源放置在 `assets/` 下，避免重复。
- 提交信息：请尽量使用简洁描述，说明新增或修改的页面内容。

本仓库仅用于页面归档与索引，不涉及特定许可声明。各页面内容版权归其原作者所有。
